# Mbizzica - Paste Sharing App

Web application built with Laravel that allows users to create, share and manage text content, similar to Pastebin.

## 🚀 Features

- User authentication (login, registration)
- Two-factor authentication (2FA)
- Create and manage text "pastes"
- Public, private and unlisted visibility
- Password protected content
- Search and filtering
- Tags and categories
- File upload support
- Comments and voting system
- REST API
- Notifications system

## 🛠️ Tech Stack

- PHP (Laravel)
- MySQL
- Blade
- Bootstrap
- JavaScript
- AJAX
- Git
